## Run Locally

Clone the project

```bash
  git clone https://github.com/Shoeb1989/LiveNews-React-Firebase-API.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Automatically fix the issues

```bash
  npm audit fix
```
To run the React project.

```bash
  npm start
```


## Compo Used

VS-CODE

**Styling:** Tailwind CSS, DaisyUI
